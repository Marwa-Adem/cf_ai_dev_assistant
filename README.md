# Cloudflare AI Developer Assistant

An AI-powered web application built on Cloudflare AI app assignment that allows users to chat with a developer assistant AI.

## Tech Stack
- Cloudflare Workers
- Workers AI (Llama 3.3)
- Cloudflare Pages
- JavaScript / HTML / CSS

## Features
- Chat-based user interface
- AI responses powered by Llama 3.3
- Basic conversation memory (last 5 messages)
- Deployed on Cloudflare

## Running Locally
1. Install Wrangler: `npm install -g wrangler`
2. Clone the repo
3. Run `wrangler dev`
